
## Installation Requirements
* Operating System: 
	* Linux
	* Windows
* Webserver:
	* Apache
	* Nginx
* PHP: >=8.0
	>Recommended: 8.0.10
* Database:
	* MySQL >=8.0
	>Recommended: 8.0
	* MariaDB >=10.2
	>Recommended: 10.5

### Deployment Notes

#### Unsupported Environments
* Free hosting of any kind
* XAMPP

#### Important Notes
* The OpenCAD Development team **DOES NOT** recommend the utilization of "free" webhosting solutions due to a multitude of reasons.
